- 新增“自动推送目标工具”：`我的skills` 可记住自动推送目标，新导入、GitHub 安装、从工具拉取和仓库收藏导入的 Skill 在入库后会自动推送到选中的工具；开启某个工具时还会立刻把当前库中已有 Skills 补推送过去。
- 新增“开机自启”设置，并打通 macOS / Windows 系统登录项同步；应用启动和设置保存时都会做幂等 reconcile，尽量保证 UI 配置与系统真实状态一致。
- 云备份页新增“最近一次同步完成时间”，可以直接看到最近一次成功备份或恢复的完成时间。
- 优化发布构建体积：默认构建加入 `-trimpath -ldflags "-s -w"`，并支持按需编译云备份 provider；Google Cloud Storage 实现切换为更轻量的 REST 客户端。
- 修复“我的提示词”在部分桌面环境下一键复制无响应的问题，新增多级剪贴板回退链路和前端回归测试。
- 修复开机自启在 macOS / Windows 下的状态对齐问题：缺失登录项不再误判失败，已开启时会刷新为当前可执行文件路径；若系统层应用失败，会回滚设置值，避免配置和真实状态错位。
- 修复自动备份调度竞态：云备份未启用时不再启动后台自动备份 goroutine，避免在退出或测试清理阶段把 `config_local.json` 异步写回本地目录。

## perf: 重构视图缓存与后台页面内存收缩
date: 2026-03-11
- 后端新增 `core/viewstate` 本地派生缓存层、性能计时助手和启动任务错峰调度，`ListSkills()`、工具推送存在索引和 `ListAllStarSkills()` 改为快照优先与增量重建，降低启动与页面重入时的磁盘扫描和首屏竞争。
- 前端移除 `SkillStatusStrip` 逐卡 `ResizeObserver` 测量，页面切换改为轻量淡入，大列表超过 18 项时关闭逐卡动画，并统一补上 Wails 事件解绑，减少切页抖动、动画掉帧和后台监听器累积带来的 CPU / memory 开销。
- 新增后台活动状态机与 `app.window.visibility.changed` 事件；窗口隐藏或不活跃约 30 秒后会卸载当前路由页面树以释放页面局部 React 状态，恢复激活后自动重新挂载并刷新当前页面数据。
- 同步补充 Go / 前端回归测试、README、features、architecture、config 文档，以及性能重构与后台内存收缩计划文档，明确本地派生缓存、后台任务错峰和后台内存收缩行为。

## perf: 引入 helper 托管的窗口生命周期
date: 2026-03-11
- 引入单入口的 helper/UI 隐藏双角色启动流程：helper 持有单实例、托盘/菜单栏与 UI 重开能力，UI 进程关闭时真正退出，从而释放 Wails/WebKit 相关进程。
- 新增本地回环控制通道、helper 实例锁、托盘控制抽象和 UI 控制服务，支持 `show-ui`、`show`、`hide`、`quit` 等控制命令，并把 Windows/macOS 托盘逻辑从直接操作 Wails 窗口改为操作 helper/UI 生命周期。
- 修复 `make` / `wails build` 在 bindings 生成阶段误触 helper bootstrap 的问题，新增 `bindings` 模式分支和测试，避免构建过程碰运行时锁文件和本地控制路径。
- 更新 README、features、architecture 中英文文档，说明 helper 承载的托盘/菜单栏重开、窗口关闭语义和 UI 控制生命周期。
